# 🚀 Digital Complaint Management System

## 📌 Project Description

This project is a **Digital Complaint Management System** designed for colleges to streamline the complaint process.
It allows students to submit complaints, track their status, and enables administrators to manage and resolve them efficiently.

👉 The system improves transparency and reduces manual work in complaint handling.

---

## 🎯 Key Features (Demo Focus)

### 👨‍🎓 Student Role

* Submit complaints with category and description
* Track complaint status (Pending, In Progress, Resolved)

### 👨‍💼 Admin Role

* View all complaints
* Update complaint status
* Provide responses to students

---

## ⚙️ Features (Full System)

* JWT-based Authentication
* Complaint Management System
* Feedback System
* File Upload (images/documents)
* AI Chatbot (Google Gemini API)
* Notifications System
* Responsive UI with Tailwind CSS

---

## 🛠️ Tech Stack
